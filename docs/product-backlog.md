# Product Backlog

## P0

- 用户注册与登录
- 课程知识库创建、编辑、删除和列表查看
- 章节维护
- 课程资料上传与资料信息维护
- PDF 文本解析
- 多条件检索与全文检索
- Agent 知识包导出
- 导出记录保存

## P1

- 课程前置关系维护：支持为一门课程设置前置课程，例如复习“数据结构与算法”时自动带入“C 语言程序设计”知识库。
- 关联课程导出：生成知识包时可选择是否包含前置课程、关联课程或后续课程的重点内容。
- 教师画像维护：记录出题老师、常见题型、评分习惯、重点章节、避坑内容等个性化信息。
- 个性化复习生成配置：根据考试目标、难度、输出类型、是否包含前置课程等条件生成复习资料。
- AI 接口配置：预留 OpenAI 兼容接口、模型名称、base URL 和 API Key 标识。
- AI 生成任务记录：保存生成提示词、任务状态、生成结果路径和失败原因。
- 导出预览
- 按章节、资料类型、重点标记筛选导出
- 课程资料统计
- 搜索记录保存
- 导出模板选择

## P2

- 批量上传
- 自动摘要
- 自动关键词提取
- 相似资料检测
- 知识包版本管理
- 基于教师画像的模拟题生成
- 基于前置课程的知识缺口提示
- 按出题风格生成冲刺复习提纲
