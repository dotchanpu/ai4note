## P1

- 课程前置关系维护：支持为一门课程设置前置课程，例如复习“数据结构与算法”时自动带入“C 语言程序设计”知识库。
- 关联课程导出：生成知识包时可选择是否包含前置课程、关联课程或后续课程的重点内容。
- 教师画像 AI 分析：系统根据老师课件、往年试题、实验要求、评分标准等资料，自动归纳出题风格、常见题型、评分偏好和重点章节。
- 教师画像证据追溯：教师画像中的判断需要关联来源资料，记录来自哪份课件、哪套真题或哪份实验要求。
- [x] 真题知识点映射：系统从往年真题中拆分题目，并将题目关联到章节和知识点，用于统计高频考点。
- 知识缺口检测：系统结合当前课程、前置课程、用户掌握度和真题高频知识点，生成薄弱知识清单。
- 教师画像人工修正：用户可以审核 AI 生成的教师画像，并在必要时修改或确认。
- 个性化复习生成配置：根据考试目标、难度、输出类型、是否包含前置课程、教师画像、知识缺口等条件生成复习资料。
- 多供应商 AI 接口配置：在已接入 DeepSeek 的基础上，继续支持数据库配置模型、base URL 和 API Key 环境变量别名。
- AI 生成任务记录：保存教师画像分析、真题映射、知识缺口检测、复习资料生成等任务的提示词、状态、结果路径和失败原因。
- 导出预览
- [x] 按章节、资料类型、重点标记筛选导出
- 课程资料统计
- 搜索记录保存
- [x] 导出模板选择
